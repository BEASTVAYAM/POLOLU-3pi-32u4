# 📺 OLED Demo (Pololu 3pi+)

## 📌 Description
This program shows how to use the OLED display on the Pololu 3pi+ robot.

---

## ⚙️ Working
- Initializes OLED display  
- Prints text at specific positions  

---

## ▶️ How to Run
1. Open DemoForOLEDVersion.ino  
2. Upload to robot  

---

## 🧩 Library
- Pololu3piPlus32U4  

---

## 📷 Output
Text appears on OLED screen.
