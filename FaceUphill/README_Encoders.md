# 🔄 Encoder Demo (Pololu 3pi+)

## 📌 Description
This program demonstrates how to read wheel encoder values.

---

## ⚙️ Working
- Reads left and right encoder counts  
- Displays or logs movement  

---

## ▶️ How to Run
1. Open Encoders.ino  
2. Upload code  
3. Move robot wheels  

---

## 🧩 Library
- Pololu3piPlus32U4  

---

## 📷 Output
Encoder counts change as wheels rotate.

---

## 🚀 Improvements
- Use for distance measurement  
- Integrate with motor control
