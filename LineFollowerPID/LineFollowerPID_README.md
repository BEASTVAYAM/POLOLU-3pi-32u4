# PID Line Follower

## Overview
Implements a PID-controlled line follower for smooth and accurate navigation.

## Features
- PID control (P, I, D)
- Smooth corrections
- High accuracy

## How It Works
- Calculates error from center
- Applies PID correction
- Adjusts motor speeds dynamically

## Usage
1. Upload code
2. Calibrate sensors
3. Place robot on track

## PID Tuning
- Increase P → faster response
- Increase D → less oscillation
- Increase I → fix drift
