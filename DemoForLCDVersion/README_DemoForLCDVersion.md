# 📟 LCD Demo (Pololu 3pi+)

## 📌 Description
This example demonstrates how to use the LCD display on the Pololu 3pi+ robot.

---

## ⚙️ Working
- Initializes LCD display  
- Prints messages on screen  
- Updates display content dynamically  

---

## ▶️ How to Run
1. Open DemoForLCDVersion.ino  
2. Connect robot  
3. Upload code  

---

## 🧩 Library
- Pololu3piPlus32U4  

---

## 📷 Output
Text is displayed on the LCD screen.

---
